Opening runtime

Checking for the orgasmic CLI.

On the host Mac: enable LAN in ~/.orgasmic/config.yaml (bind_host: 0.0.0.0, lan_enabled: true), restart the daemon, and copy the token from ~/.orgasmic/user/auth/token. On another network, point this at a tunnel URL (e.g. cloudflared) or a Tailscale address.
