    How do I find these?

    On the computer running the orgasmic daemon, enable LAN access in ~/.orgasmic/config.yaml — set bind_host: 0.0.0.0 and lan_enabled: true, then restart the daemon.

    Copy the bearer token from ~/.orgasmic/user/auth/token (same path on macOS, Linux, and Windows).

    Off the local network, point the URL at a tunnel (e.g. cloudflared) or a Tailscale address instead of the LAN IP.
