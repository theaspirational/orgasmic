Architecture graph policy:
- Prefer functional component nodes with clear interfaces and source paths.
- Use typed edges where possible: READS, WRITES, EXPOSES_REST, EXPOSES_WS,
  SUBSCRIBES_TO, SPAWNS, CALLS.
- Use DEPENDS_ON only when no more precise edge applies.
