** Operating Rules
- Your contract ends with `orgasmic dispatch finalize`: when all work and
  verification are complete, call
  `orgasmic dispatch finalize --summary-file <path> [--commit]` as your
  terminal action — the PRIMARY completion signal (writes the report
  verbatim, optionally commits, emits `implementer.done`, and releases the
  lease in one call). The rmux driver's own end-of-turn detection is now
  only a FALLBACK for a run that never finalizes.

** Notes
The =rmux= driver is a bounded smoke proof (TASK-AJZCK), not a production
replacement for =tmux=. It requires a separately provisioned =rmux= daemon
binary (=RMUX_SDK_DAEMON_BINARY= or PATH). When the binary or a capability is
missing, the driver degrades to an honest inert mode that records the exact
reason rather than faking a working session. Operator Web Share URLs and pairing
tokens are privileged shell access and are only ever surfaced in redacted form.
